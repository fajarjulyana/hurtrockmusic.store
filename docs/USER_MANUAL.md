## Membuat Akun dan Login

### Mendaftar Akun Baru:

1. **Akses Halaman Register:**
   - Klik tombol "Login/Register" di header website
   - Pilih tab "Register" jika belum memiliki akun

2. **Isi Form Registrasi:**
   ```
   Required Information:
   ├── First Name (Nama Depan) *
   ├── Last Name (Nama Belakang) *
   ├── Email Address *
   └── Password (min. 6 karakter) *
   ```

3. **Submit dan Verifikasi:**
   - Klik tombol "Register"
   - Sistem akan otomatis login setelah registrasi berhasil
   - Anda akan diarahkan ke dashboard atau homepage

### Login ke Akun Existing:

1. **Akses Halaman Login:**
   - Klik tombol "Login" di header
   - Atau kunjungi halaman login langsung

2. **Masukkan Credentials:**
   - Email address yang terdaftar
   - Password akun Anda

3. **Fitur Login:**
   - Remember me (session akan bertahan lebih lama)
   - Forgot password (jika tersedia)
   - Auto-redirect ke halaman sebelumnya

### Keamanan Akun:
- **Password Requirements**: Minimal 6 karakter dengan kombinasi huruf dan angka
- **Session Management**: Otomatis logout setelah periode inactive
- **Secure Cookies**: Data session disimpan dengan aman
- **Password Hashing**: Password di-encrypt dengan teknologi bcrypt

## Enhanced FAQ

### Search & Navigation:

**Q: Bagaimana cara menggunakan advanced search?**
A: Gunakan search box di header untuk basic search, atau buka Products page untuk access ke advanced filters seperti category, price range, rating, dan status.

**Q: Kenapa search results tidak akurat?**
A: Search engine mencari di product name, description, dan category. Coba gunakan keywords yang lebih spesifik atau gunakan filters untuk mempersempit hasil.

**Q: Bagaimana cara save search results?**
A: Bookmark halaman dengan filters applied, atau save individual products ke wishlist untuk referensi nanti.

### Chat Support:

**Q: Kapan live chat tersedia?**
A: Live chat tersedia during business hours (Senin-Jumat 09:00-17:00 WIB). Outside hours, Anda bisa leave message dan admin akan respond via email.

**Q: Bisakah attach files di chat?**
A: File sharing feature sedang development. Untuk sekarang, describe issue secara detail atau send files via email ke support@hurtrockstore.com.

**Q: Chat history tersimpan berapa lama?**
A: Chat history tersimpan permanently di account Anda untuk reference. Anda bisa access previous conversations di dashboard.

### Shopping & Orders:

**Q: Bagaimana cara track order real-time?**
A: Login ke account, buka Order History di dashboard, dan klik order untuk detailed tracking information dengan status updates.

**Q: Bisakah modify order setelah checkout?**
A: Order modification hanya bisa dilakukan sebelum status "Processing". Contact customer support via chat atau phone untuk assistance.

**Q: Apa itu persistent cart?**
A: Cart items tersimpan di account Anda dan akan tetap ada bahkan setelah logout/login atau ganti device.

### Mobile & Technical:

**Q: Bagaimana cara install PWA di mobile?**
A: Buka website di mobile browser, tap "Add to Home Screen" option di browser menu. Icon Hurtrock akan appear di home screen.

**Q: Website lambat loading di mobile?**
A: Clear browser cache, ensure stable internet connection, atau try accessing during off-peak hours. PWA mode biasanya lebih fast.

**Q: Fitur apa yang available offline?**
A: Product catalog browsing dan order history available offline di PWA mode. Checkout dan real-time features require internet.
